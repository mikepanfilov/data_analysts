Добрый день! Я - инструкция по запуску дашборда для телеком сервиса "Нупозвони".

Работа выполнена на Яндекс.Облаке - работающая версия дашборда доступна по адресу http://84.201.181.144:8050/

1) Базы данных:

Датасет telecom_dataset.csv помещен в подпапку datasets на сервере и содержит колонки:

- user_id — Идентификатор клиентского аккаунта в сервисе
- date — Дата статистики
- direction — Направление вызовов (out - исходящий вызов, in — входящий вызов)
- internal — Является ли звонок внутренним звонком между операторами клиента
- operator_id — Идентификатор оператора
- is_missed_call — Является ли звонок пропущенным
- calls_count — Количество звонков
- call_duration — Длительность звонка (без учета времени ожидания)
- total_call_duration — Длительность звонка (с учетом времени ожидания)


2) Дашборд:

В дашборде содержится два набора данных:

1. Набор №1
    1. Постройте гистограмму распределения длительности звонков
    2. Постройте круговую диаграмму, отображающую соотношение кол-ва внутренних и внешних вызовов
    3. Добавьте фильтр дашборда по направлению вызовов
2. Набор №2
    1. Постройте диаграмму количества вызовов по дням
    2. Постройте круговую диаграмму, отображающую соотношение кол-ва входящих и исходящих вызовов
    3. Добавьте фильтр дашборда по типу вызова (внешний или внутренний)

Скрипт дашборда расположен в файле telecom_dash.py.

Чтобы запустить дашборд нужно запустить в командной строке:

python3 telecom_dash.py

Дашборд будет работать как на локальной машине по адресу localhost:8050 , 
так и с внешнего источника по адресу http://<ваш внешний IP>:8050

Дополнительные библиотеки, необходимые для запуска находятся в файле requirements.txt
Установить их можно следующей строчкой кода в терминале:

pip3 install -r requirements.txt
